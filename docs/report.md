# Relatorio Final - Sistema de Notas TDD

## Resumo

Projeto desenvolvido para a disciplina de Teste de Software seguindo a metodologia TDD.
Sistema de gerenciamento de notas academicas com 4 classes de dominio, 53 testes automatizados
e 5 regras de negocio validadas.

## Tecnologias

- Java 17, JUnit 5, Maven

## Resultados

- 53/53 testes passando
- 17 metodos testaveis
- 5 regras de negocio implementadas
- 4 ciclos Red-Green-Refactor completos
